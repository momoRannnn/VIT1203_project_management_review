# 项目管理刷题工具

生成时间：2026-06-29（更新于 2026-06-30）

## 快速使用

浏览器直接打开 `index.html` 即可使用，无需服务器或构建工具。

## 文件说明

- `index.html` — 应用入口，包含所有 CSS 样式和 HTML 结构
- `app.js` — 主协调器：初始化、事件绑定、数据流程控制
- `state.js` — 状态管理与数据访问：localStorage 读写、题目查询、统计计算
- `memory.js` — 间隔重复逻辑：记忆曲线、到期/薄弱判定
- `quiz.js` — 刷题引擎：出题池构建、作答流程、交卷与成绩
- `ui.js` — 所有 DOM 渲染：章节选择器、记忆曲线面板、趋势图、热力图、知识图谱
- `questions.js` — 题库数据（120 道期末真题），由原始结构化数据生成

## 功能特性

1. **六种刷题模式** — 随机刷题、按章节、到期复习、错题重练、按知识点、收藏题目
2. **两套题库** — 全部 120 题 / 第一套 Q01–Q60 / 第二套 Q61–Q120，可在设置面板切换
3. **作答隐藏** — 做题时只显示题干和选项，不显示答案、章节、出处或解析
4. **交卷解析** — 交卷后显示正确答案、你的答案、章节、知识点、记忆点、易错点和出处
5. **记忆曲线** — 答错 10 分钟后复习；连续答对后按 1、3、7、14、30、60 天间隔复习
6. **错题本** — 按错误次数排序，支持多选针对性练习，点击预览题目详情
7. **薄弱知识点分析** — 从错题自动聚合薄弱知识点，支持一键针对性练习
8. **收藏功能** — 收藏重点题目，支持收藏模式刷题
9. **全局搜索** — 按题干关键词、知识点、章节实时搜索，点击结果直接练习
10. **做题趋势图** — Chart.js 折线图展示正确率随时间变化
11. **掌握度热力图** — 章节 × 知识点网格，颜色编码掌握程度，悬浮查看详情，点击直达练习
12. **知识图谱** — D3 力导向图展示知识点关联，支持拖拽节点、滚轮缩放、点击查看关联题目

## 数据保存

练习记录保存在当前浏览器的 `localStorage` 中（key: `pm_quiz_tool_v2`），不会上传。可导出/导入 JSON 数据进行备份。

## 技术栈

- 纯前端 vanilla JS，无构建工具依赖
- D3.js v7 — 知识图谱力模拟
- Chart.js v4 — 做题趋势折线图
- CSS Grid / Flexbox 响应式布局
- localStorage 持久化 + 间隔重复算法
