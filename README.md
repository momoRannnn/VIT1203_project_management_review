# 项目管理刷题工具

生成时间：2026-06-29

## 文件说明

- `index.html`：刷题工具入口，浏览器打开即可使用。
- `app.js`：刷题、按章节练习、错题重练、记忆曲线和统计逻辑。
- `questions.js`：由 `questions.json` 生成的浏览器可直接加载题库数据。
- `questions.json`：60 道期末真题原始结构化数据。
- `knowledge_points.json`、`knowledge_graph.json`：历史生成数据，当前刷题页面不再加载。

## 当前功能

1. 随机刷题：可设置题数、限时、是否打乱题目和选项。
2. 按章节刷题：可选择一个或多个章节范围。
3. 作答隐藏：做题时只显示题干和选项，不显示答案、章节、出处或解析。
4. 交卷解析：交卷后显示正确答案、你的答案、章节、考点、记忆点、易错点和出处。
5. 记忆曲线：答错 10 分钟后复习；连续答对后按 1、3、7、14、30、60 天间隔复习。
6. 统计记录：本地记录总体正确率、章节正确率、到期复习题和错题。

## 数据保存

练习记录保存在当前浏览器的 `localStorage` 中，不会上传。
